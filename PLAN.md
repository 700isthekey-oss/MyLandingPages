# BALLET Website Prototype

Version: 1.0

# Project Overview

BALLET는 발레에서 영감을 받은 프리미엄 여성 애슬레저 브랜드의 시범 웹사이트 프로젝트이다.

본 프로젝트의 목적은 실제 판매가 아닌 브랜드 방향성, 디자인 시스템, 사용자 경험(UI/UX), 카피라이팅을 검증하는 것이다.

향후 브랜드명, 제품 라인업, 생산 및 물류 체계가 확정되면 실제 결제 기능을 포함한 정식 쇼핑몰로 확장한다.
